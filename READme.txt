All matlab files can be run, model building parts commented out for convenience.

Comparison of random forest and naive bayes.pdf - PDF of poster comparing models

coursework.m - Contains all code of initial analysis as well as code producing histograms and ROC curve

comparison_graph.m - Code that generates graph comparing average accuracy and time

RF_final_model.m - Contains code that produces all results for random forest

NB_final_model.m - Contains code that produces all results for naive bayes

NB_final_model.mat - Best naive bayes model saved as a variable

RF_final_model.mat - Best random forest model saved as a variable

Test sets saved as Xtest.mat for predictor variables and Ytest.mat for target variables
